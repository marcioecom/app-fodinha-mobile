# App to register matches and points

### Initial idea
There was a idea of a app to easily handle points of a game named "Fodinha" (i knew by my cousin)

### What i needed
- Have 5 points to each player and decrement them
- A matches history

### Technologies I've used
- React Native
- Expo
- Typescript
- Google Fonts
